# 💳 3D Mastercard Clone

A visually realistic Mastercard-inspired credit card built using pure HTML and CSS. The project features a smooth 3D flip animation, front and back card faces, and carefully crafted details inspired by modern payment cards.

---

## ✨ Features

- Realistic Mastercard-inspired design
- Front and back card faces
- Smooth 3D flip animation
- Perspective-based transforms
- Custom EMV chip design
- Magnetic strip and CVV section
- Responsive card layout
- Pure HTML and CSS implementation
- No JavaScript used

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Remix Icons

---

## 📚 Concepts Practiced

### CSS 3D Transforms
- `perspective`
- `transform-style: preserve-3d`
- `rotateY()`
- `backface-visibility`

### Animations
- CSS transitions
- Hover interactions
- Multi-face transformations

### Layout Techniques
- Flexbox
- Absolute positioning
- Nested components

### Styling Techniques
- Custom card design
- Layer management using `z-index`
- Gradient effects
- Responsive sizing

---

## 🎯 Project Goal

The purpose of this project was to explore CSS beyond traditional layouts and experiment with creating interactive, realistic UI components using only HTML and CSS.

Rather than relying on external libraries, the focus was on understanding how 3D transforms work and how multiple elements can be combined to create immersive effects.
